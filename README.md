# viewer-updates
Repository containing the viewer's velopack updates.  This includes various public facing viewer channels.

We utilize this for velopack updates.  The Viewer Version Manager passes the viewer a blob of LLSD containing the location of the appropriate JSON file located in this repository, that velopack then downloads and updates against.

We provide this to the public to see what specific updates we have available, and as a reference point for implementing your own viewer's auto update functionality on top of velopack.
